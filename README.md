A modern, fully responsive, and elegant landing page website designed for a professional carpet cleaning service. This project showcases premium cleaning services, interactive booking options, and a clean user experience.


🚀 Live Demo
💻 https://diamond-carpetcleaning.co.uk/

---

## ✨ Features

*   **Premium UI/UX:** Sleek, diamond-themed color palette representing cleanliness and high quality.
*   **Fully Responsive:** Optimized for all screen sizes (Mobile, Tablet, and Desktop).
*   **Our Services:** Interactive section displaying various services (Carpet Cleaning, Rug Care, Sofa & Upholstery Cleaning, Stain Removal).
*   **Quick Booking / Contact Form:** A user-friendly form for clients to easily request a quote or book a service.
*   **Testimonials Section:** Customer reviews to build trust and credibility.
*   **FAQ Section:** Frequently asked questions about pricing, process, and timing.

---

## 🛠️ Tech Stack Used

*   **HTML5:** Semantic markup for structured and SEO-friendly content.
*   **CSS3:** Custom styling, variables, grid, flexbox, and smooth hover animations.
*   **JavaScript (ES6):** Responsive navigation menu, smooth scrolling, and form validation.
